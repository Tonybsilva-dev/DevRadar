<h1>DevRadar</h1>
<span>An application that facilitates the search for developers in your region</span>


<h3>Starting the application:</h3>
<h6>Backend</h6>

```
yarn dev # Inicializa o servidor
```

<h6>Frontend</h6>

```
yarn start # Inicializa o React
```

<h6>Mobile</h6>

```
yarn start # Inicializa o expo
```
